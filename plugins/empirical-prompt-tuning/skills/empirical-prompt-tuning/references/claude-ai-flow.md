# Claude.ai 환경에서의 dispatch 절차

Task tool이 없는 환경. 단일 세션 직렬 폴백 또는 사용자 협력으로 외부 세션 분리.

## 전제

- Claude.ai 웹/모바일/데스크톱 앱
- 서브에이전트 dispatch 불가
- 한 세션 내에서 평가 = 글쓴이 = 판정자가 동일하므로 편향이 들어간다는 한계 인지

## 기본 전략 (3가지 중 택일)

### 전략 A: 사용자 협력 외부 세션 분리 (권장)

가장 재현성 높음. 글쓴이 본인이 별도 Claude.ai 세션 또는 다른 LLM(ChatGPT, Gemini 등)을 띄우고 dispatch 프롬프트를 복사·붙여넣기로 실행시킨다.

절차:
1. Claude는 `assets/dispatch-prompt-template.md`를 채운 완성형 프롬프트를 사용자에게 제공
2. 사용자가 별도 세션(새 Chrome 탭의 Claude.ai 신규 대화 / 시크릿 창 / 다른 AI)에 붙여넣고 실행
3. 사용자가 결과 보고서를 원래 세션에 가져와 Claude에게 전달
4. Claude는 정성·정량 데이터를 추출하여 [5] 단계 진행

이 방법은 매번 신규 dispatch 원칙을 지킨다.

### 전략 B: 단일 세션 내 롤플레이 격리 (편향 위험)

같은 Claude.ai 세션에서 진행하되, 평가 단계에서 명시적 컨텍스트 격리 지시.

```
지금부터 당신은 위에 있는 SKILL.md를 한 번도 본 적 없는
별도의 Claude 인스턴스입니다. 위 작성 과정의 모든 내용은
잊고, 아래 시나리오만을 보고 SKILL.md에 따라 작업하십시오.
작성자의 의도를 추측하지 말고, 본문에 명시되지 않은 것은
"불명확점"으로 보고하십시오.

[SKILL.md 본문]
...

[시나리오]
...

[요건 체크리스트]
...
```

한계: 같은 세션이므로 학습 편향이 완전히 빠지지 않는다. 사용자에게 이 한계를 명시하고, 결과를 "참고용"으로 다룬다.

### 전략 C: 평가 자체를 명시 보고 (최후 수단)

별도 dispatch 환경 확보가 불가능하면, 평가를 시도하지 말고 명시적으로 "평가 단계 건너뜀"을 보고한다. **자기 재독으로 대체하지 않는다.** 자기 재독은 편향이 가장 큰 방법이므로 무평가보다 나쁘다.

## 단일 세션 직렬 절차 (전략 B 사용 시)

1. 사용자에게 평가 대상 SKILL.md 본문 제시 요청
2. Claude가 시나리오 3개(median + edge + hold-out) 설계, 사용자 승인
3. 요건 체크리스트 작성, `[critical]` 태그 부여, 사용자 승인
4. 시나리오마다 별도 메시지로 진행 (한 메시지에 시나리오 1개만 처리)
5. 각 시나리오마다 다음 형식으로 응답:

```
[시나리오 X]

산출물:
<생성물 또는 실행 결과 요약>

요건 달성:
1. [critical] ... ○
2. ...

불명확점:
- ...
- ...

재량 보완:
- ...

재시도: <횟수>회 (이유: ...)

[메트릭 - 단일 세션 모드]
- 시나리오 처리에 사용한 도구 수: <counted>
- 응답 생성 시간: 측정 불가
```

6. 3개 시나리오 모두 완료 후 종합 보고서 생성
7. [6] 단계로 진행

## 한계 명시

전략 B 사용 시 사용자에게 다음을 명시:

> 이 평가는 동일 세션 내에서 진행되었으므로 글쓴이-판정자 분리가 완전하지 않습니다. 결과는 참고용이며, 가능하다면 별도 세션에서 dispatch-prompt-template.md를 사용한 외부 평가(전략 A)를 권장합니다.

## Artifacts 활용

Claude.ai의 Artifacts에 다음을 분리하여 저장:
- 평가 대상 프롬프트 (변경되지 않음, reference용)
- 시나리오 + 요건 체크리스트 (반복 간 고정)
- 반복별 결과 로그 (반복마다 누적)

이를 통해 사용자가 반복 간 추이를 한눈에 볼 수 있다.

## 사용자에게 안내해야 할 사항

평가 시작 전 다음을 사용자에게 알린다:

```
이 환경(Claude.ai)에서는 별도 서브에이전트 dispatch가 불가능합니다.
다음 3가지 옵션 중 선택해 주세요:

A. 새 Claude.ai 세션 또는 다른 AI를 직접 띄워 평가 (권장, 재현성 ↑)
   → 제가 dispatch 프롬프트를 만들어 드릴 테니 복사해서 사용하세요
B. 이 세션 내에서 롤플레이 격리로 진행 (편향 위험, 결과는 참고용)
C. 평가 단계 건너뛰고 본문 분석만 진행
```
